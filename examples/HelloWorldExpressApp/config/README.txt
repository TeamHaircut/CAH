The config folder is where you keep your Express application configuration
files. In time you'll add more modules to your application and each module
will be configured in a dedicated JavaScript file, which is placed inside this
folder. Currently, it contains several files and folders, which are as follows:
° The env folder is where you'll keep your Express application
environment configuration files
° The config.js file is where you'll configure your Express application
° The express.js file is where you'll initialize your Express application
